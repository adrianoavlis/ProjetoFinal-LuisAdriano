# Plataforma para Análise de Produtos da Cesta Básica

## Introdução

Este repositório contém os recursos, códigos e documentação relacionados ao meu Trabalho de Conclusão de Curso (TCC), cujo objetivo é a implementação de uma plataforma de análise de valor voltada para a comunidade da Rocinha, com foco na comercialização de produtos da cesta básica. O projeto visa atender a uma demanda latente na comunidade, oferecendo uma solução inovadora que não apenas facilitará o comércio local, mas também proporcionará insights valiosos sobre o relacionamento entre consumidores e produtos essenciais.

## Funcionalidades

1. **Catálogo Online de Produtos**
   - Exibição organizada dos produtos essenciais, facilitando a análise dos itens mais importantes para o usuário.

2. **Dashboard Analítico**
   - Recurso de análise para identificar padrões de consumo, produtos mais vendidos e análise de preferências segmentadas.

3. **Registro de Preços**
   - Histórico de preços de produtos, permitindo que lojistas e consumidores acompanhem a variação dos preços ao longo do tempo.

4. **Filtro por público**
   - Dados personalizados sobre os produtos mais vendidos através de filtros sobre público alvo, ajudando lojistas a focar em produtos específicos.

5. **Extração de Dados**
   - Coleta de informações em marketplaces externos para complementar os dados, aplicando scrapping e garantindo uma visão de mercado detalhada e relevante.

## Tecnologias 

- **Backend:** Java 8 
- **Banco de Dados:** MySQL 

## ⚙️ Instalação e Execução

   ### Pré-requisitos
   
   - JDK 8 instalado
   - MySQL 8.0 configurado
   - Apache Tomcat 9 instalado
   - Maven instalado
   - ChromeDriver instalado no sistema
